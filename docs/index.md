# OSH 知識庫

這裡彙整職業安全衛生相關的文件與圖示，並以 MkDocs 發佈為靜態網站。

## 主要主題

- 管理與制度： [安全衛生管理員](安全衛生管理員.md)、[安全衛生組織](安全衛生組織.md)、[職災補償](職災補償.md)、[通報 / 備查](通報_備查.md)、[法規參考連結](法規參考連結.md)
- 作業： [作業總覽](作業/作業.md)、[局限空間作業](作業/局限空間作業.md)
- 危害： [危害總覽](危害/危害.md)、[化學品](危害/化學品.md)、[物理性危害](危害/物理性危害.md)
- 設備與設施： [機械設備](機械設備.md)、[營造設施](營造設施.md)
- 其他： [計劃書](計劃書.md)、[分級管理](分級管理.md)、[指引 + CNS](指引 + CNS.md)

## 使用方式（快速操作）

1. 安裝套件：`pip install -r requirements.txt`
2. 本地預覽：`mkdocs serve` 後開啟 `http://127.0.0.1:8000/`
3. 正式輸出：`mkdocs build --clean --strict` 確認鏈結與導航正確

## 結構

-   `docs/`：MkDocs 原始文件（含圖檔）。
-   `python/`：Python 程式與輔助腳本。
-   `site/`：MkDocs build 後的靜態網頁輸出。
