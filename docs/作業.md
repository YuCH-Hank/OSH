## 特殊危害作業 休息時間

對於勞工具有特殊危害之作業，亦應規定減少勞工工作時間，並在工作時間中予以適當之休息

### 高溫

高溫作業勞工如為連續暴露達 1 小時以上者，以每小時計算其暴露時量平均綜合溫度熱指數，
間歇暴露者，以 2 小時計算其暴露時量平均綜合溫度熱指數，
並依下表規定，分配作業及休息時間。

|            |          |          |          |          |
| ---------- | -------- | -------- | -------- | -------- |
| 輕 工 作   | 30.6     | 31.4     | 32.2     | 33.0     |
| 中度工作   | 28.0     | 29.4     | 31.1     | 32.6     |
| 重 工 作   | 25.9     | 27.9     | 30.0     | 32.1     |
| 時間比例   |          | 25％休息 | 50％休息 | 75％休息 |
| 每小時作息 | 連續作業 | 75％作業 | 50％作業 | 25％作業 |

輕工作：指僅以坐姿或立姿進行手臂部動作以操縱機器者。
中度工作：指於走動中提舉或推動一般重量物體者。
重工作：指鏟、掘、推等全身運動之工作者。

### 異常氣壓作業 (特別危害健康作業)

### 高架作業

每連續作業二小時，應給予作業勞工下列休息時間：

一、高度在二公尺以上未滿五公尺者，至少有二十分鐘休息。
二、高度在五公尺以上未滿二十公尺者，至少有二十五分鐘休息。
三、高度在二十公尺以上者，至少有三十五分鐘休息。

### 精密作業

雇主使勞工從事精密作業時，應縮短工作時間，於連續作業二小時，給予作業勞工至少十五分鐘之休息。

### 重體力勞動

重體力作業時，應綜合考量勞工體能負荷，減少其作業時間，給予每小時至少 20 分鐘之充足休息

## 作業主管

|          | 作業主管                     |
| -------- | ---------------------------- |
| 屋頂作業 | 於易踏穿材料構築之屋頂作業時 |

作業主管於現場辦理下列事項：

1. 決定作業方法，指揮勞工作業。
2. 實施檢點，檢查材料、工具、器具等，並汰換其不良品。
3. 監督勞工確實使用個人防護具。
4. 確認安全衛生設備及措施之有效狀況。
5. 前二款未確認前，應管制勞工或其他人員不得進入作業。
6. 其他為維持作業勞工安全衛生所必要之設備及措施。

## 噪音作業 - 音源控制

1. 減少機台震動

   低頻：震動絕緣器

2. 減少衝擊
3. 減少摩擦
4. 減少共振體
5. 密封音源(高頻)
6. 利用消音器
   - 散失型消音器(吸收)
   - 反應行銷音器(抗性)

### 控制工程：傳播途徑控制

1. 吸音材料的使用

   ```text
   材料的吸音係數，通常以六個頻率的吸音係數：125，250，500, 1000, 2000, 4000 Hz 表示。

   NRC 噪音減少係數為中間四個頻率的平均數。
   ```

2. 遙控式操作

3. 設置隔音屏

   ```text
   STC（Sound Transmission Class）"噪音傳送等級"是一個用來表示裝修物件（例如牆壁、門、窗等）隔音能力的指標，表示物件隔音的效果。STC 的數字越高，隔音效果越好，且 STC 是以分貝（dB）為單位。
   ```

## 震動作業 - 震動控制

彈簧的自然頻率$f_n$與靜變形之間的關係如下：

$$
  f_{n}=\frac{5}{\sqrt{\delta_1[cm]}}[Hz]
$$

激動頻率與自然頻率比值與振動絕緣程度關係

| $\frac{f}{f_n}=R$ | 振動絕緣程度 |
| ----------------- | ------------ |
| <1.4              | 放大         |
| 1.4~3             | 可忽略       |
| 3~6               | 低           |
| 6~10              | 尚可         |
| >10               | 高           |
