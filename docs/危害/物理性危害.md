## 燒傷

| 燒傷深度 | 深度範圍                | 症狀                                 | 癒合情形                                  |
| -------- | ----------------------- | ------------------------------------ | ----------------------------------------- |
| 一度     | 表皮淺層                | 皮膚發紅、腫脹、觸痛                 | 3–5 天癒合，無疤痕                        |
| 淺二度   | 表皮+真皮表層（約 1/3） | 紅腫、水泡、劇痛                     | 約 14 天癒合，輕微或無疤痕                |
| 深二度   | 表皮+真皮深層           | 淺紅色、大水泡，感覺較鈍             | 21 天以上，留明顯疤痕，宜早期植皮避免感染 |
| 三度     | 全層皮膚                | 焦黑或蒼白，乾硬如皮革，**疼痛消失** | 需植皮，留肥厚疤痕與功能障礙              |
| 四度     | 皮膚、皮下、肌肉、骨骼  | 組織壞死呈焦炭狀                     | 需皮瓣、電療等，部分需截肢                |

## 電壓與電氣安全

![](../image/安全衛生管理員/電壓.drawio.svg)

-   **技術人員配置**：

    1. **低壓**：< 660V，且契約容量 ≥50 kW → 置**初級電氣技術人員**。
    2. **高壓**：660V ～ 22,800V → 置**中級電氣技術人員**。
    3. **特高壓**：> 22,800V → 置**高級電氣技術人員**。

-   **漏電斷路器**：**150V** 以上之移動式電動器具應設置。
