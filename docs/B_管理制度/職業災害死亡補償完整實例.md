# 📘 職業災害死亡補償完整實例

（勞基法 × 勞保條例雙軌制度）

---

## 一、制度背景：台灣的「雙軌制」

台灣對職業災害採取**雙軌保障制度**：

| 軌道   | 法源                           | 性質                     |
| ------ | ------------------------------ | ------------------------ |
| 第一軌 | 《勞動基準法》第 59 條         | 雇主的「無過失補償責任」 |
| 第二軌 | 《勞工保險條例》第 49 ～ 51 條 | 社會保險給付             |

兩者的關係是：

> **勞保先付 → 雇主負最後補足責任**

法律明文規定：
若同一事故已由勞保或其他法令給付，雇主可以「抵充」，但**不能少於勞基法應給的最低金額**。

---

## 二、關鍵法條

### ① 勞動基準法 第 59 條（職災死亡）

> 勞工因職業災害死亡時，雇主應給：

-   喪葬費：**5 個月平均工資**
-   死亡補償：**40 個月平均工資**

➡ 合計：**45 個月平均工資**

遺屬順位：

1. 配偶、子女
2. 父母
3. 祖父母
4. 孫子女
5. 兄弟姊妹

不論年齡、收入、是否工作，**只要是法定遺屬就一定要給**。

---

### ② 勞工保險條例 第 49 ～ 51 條（職災死亡）

勞保提供三種給付：

| 類型          | 給多少                             |
| ------------- | ---------------------------------- |
| 喪葬津貼      | 5 個月投保薪資（無遺屬為 10 個月） |
| 遺屬年金      | 每月 50% 投保薪資（可加成）        |
| 或 遺屬一次金 | 40 個月投保薪資                    |

---

## 三、最大關鍵差異

| 勞基法               | 勞保                 |
| -------------------- | -------------------- |
| 平均工資             | 平均月投保薪資       |
| 實際拿到的錢         | 公司申報給勞保的金額 |
| 包含加班、津貼、獎金 | 多半只報底薪         |

👉 這就是所有爭議的核心。

---

## 四、完整實務案例

### 勞工 A 的實際狀況

| 項目                       | 金額       |
| -------------------------- | ---------- |
| 本薪                       | 35,000     |
| 加班＋津貼                 | 15,000     |
| **實際月收入（平均工資）** | **50,000** |
| 公司勞保投保薪資           | **28,800** |

A 不幸發生職災死亡。

---

## 五、依法應給多少？

### ① 依《勞基法》第 59 條（雇主責任）

> 50,000 × 45 = **2,250,000 元**

這是雇主「最少」要負的責任。

---

### ② 勞保實際會給多少？

| 給付       | 計算                    |
| ---------- | ----------------------- |
| 喪葬津貼   | 28,800 × 5 = 144,000    |
| 遺屬一次金 | 28,800 × 40 = 1,152,000 |
| **合計**   | **1,296,000**           |

---

## 六、差額誰負責？

依勞基法 59 條的「抵充」規定：

> 雇主應付金額 – 勞保已給付金額 = 雇主要補的錢

2,250,000 – 1,296,000 = **954,000**

👉 這 95 萬 **由雇主補給遺屬**

---

## 七、這正是我們剛剛對話的核心

你指出：

> 「一個是平均薪資，一個是平均月投保金額」

這句話的法律意義就是：

> **公司若低報勞保 → 勞保少給
> → 雇主就要補更多**

因此低報不會讓公司省錢，
反而會在職災時爆出**巨額補償責任**。

---

## 八、一句話總結

> **勞保是幫雇主先付，
> 勞基法是用「真實薪水」兜底。**

這就是為什麼在職災死亡案件裡，
真正決定家屬能拿多少錢的，
不是勞保，
而是 **《勞動基準法》第 59 條 × 平均工資**。
