## 危險性工作場所

> [勞動檢查法 第 26 條](https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=N0070001&flno=26)

危險性工作場所，非經勞動檢查機構審查或檢查合格，事業單位不得使勞工在該場所作業:

1. 從事`石油裂解`之石化工業之工作場所。
2. `農藥製造`工作場所。
3. `爆竹煙火`工廠及`火藥`類製造工作場所。
4. 設置`高壓氣體類壓力容器`或`蒸汽鍋爐`，其壓力或容量達中央主管機關規定者之工作場所。
5. 製造、處置、使用`危險物、有害物`之數量達中央主管機關規定數量之工作場所。
6. 中央主管機關會商目的事業主管機關指定之`營造工程`之工作場所。
7. 其他中央主管機關指定之工作場所。

> [危險性工作場所審查及檢查辦法 第二條](https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=N0070019&flno=2)

-   危險性工作場所分類如下:

    | 類型 | 說明                                                                                                                                                                                                                                                                                                                                                                                         |
    | ---- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | 甲類 | （一）從事**石油**產品之**裂解**反應，以製造石化基本原料之工作場所。<br>（二）**製造、處置、使用危險物、有害物**之數量達本法施行細則附表一及附表二規定數量之工作場所。                                                                                                                                                                                                                       |
    | 乙類 | （一）使用異氰酸甲酯、氯化氫、氨、甲醛、過氧化氫或吡啶，從事**農藥**原體合成之工作場所。<br>（二）利用**氯酸鹽類**、**過氯酸鹽類**、硝酸鹽類、硫、硫化物、磷化物、木炭粉、金屬粉末及其他原料製造**爆竹煙火**類物品之爆竹煙火工廠。<br>（三）從事以化學物質製造爆炸性物品之**火藥**類製造工作場所。                                                                                           |
    | 丙類 | 指蒸汽鍋爐之傳熱面積在 `500 m²`以上(危險性設備)，或高壓氣體類壓力容器一日之冷凍能力在 `150T` 以上<br>或處理能力符合下列規定之一者：<br>（一）`1000 m³`以上之氧氣、有毒性及可燃性高壓氣體。(危險性設備)<br>（二）`5000 m³`以上之前款以外之高壓氣體。(危險性設備)                                                                                                                              |
    | 丁類 | 指下列之營造工程：<br>（一）建築物高度在 `80m` 以上之建築工程。<br>（二）單跨橋梁之橋墩跨距在 `75m` 以上或多跨橋梁之橋墩跨距在 `50m` 以上之橋梁工程。<br>（三）採用**壓氣施工**作業之工程。<br>（四）長度 `1000m` 以上或需開挖 `15m` 以上豎坑之隧道工程。<br>（五）開挖深度達 `18m` 以上，且開挖面積達 `500m²` 以上之工程。<br>（六）工程中模板支撐高度 `7m` 以上，且面積達 `330m²` 以上者。 |
