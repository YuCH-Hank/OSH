## 特殊危害作業 [^1]

對於勞工具有特殊危害之作業, 亦應規定減少勞工工作時間, 並在工作時間中予以適當之休息

1. 高溫場所 (屬特別危害健康作業, 需特殊健康檢查)
2. 異常氣壓作業 (特別危害健康作業, 需特殊健康檢查)
3. 高架作業
4. 精密作業
5. 重體力勞動

| 作業種類   | 休息時間                                                                                                                                                                                                              |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 高溫       | 依照輕中重度工作處理[^4]                                                                                                                                                                                              |
| 異常氣壓   |                                                                                                                                                                                                                       |
| 高架作業   | 每 2 小時, 應給予作業勞工下列休息時間：<br>1. 高度在 2 公尺以上**未滿**5 公尺者, 至少有 20 分鐘休息。<br>2. 高度在 5 公尺以上**未滿**20 公尺者, 至少有 25 分鐘休息。<br>3. 高度在 20 公尺以上者, 至少有 35 分鐘休息。 |
| 精密作業   | 每 2 小至少 **15** 分鐘之充足休息                                                                                                                                                                                     |
| 重體力勞動 | 每 1 小時至少 **20** 分鐘之充足休息                                                                                                                                                                                   |

## 作業主管

作業主管於現場辦理下列事項：

1. 決定作業方法, 指揮勞工作業。
2. 實施檢點, 檢查材料、工具、器具等, 並汰換其不良品。
3. 監督勞工確實使用個人防護具。
4. 確認安全衛生設備及措施之有效狀況。
5. 前二款未確認前, 應管制勞工或其他人員不得進入作業。
6. 其他為維持作業勞工安全衛生所必要之設備及措施。

## 噪音作業 - 音源控制

1. 減少機台震動

    低頻：震動絕緣器

2. 減少衝擊
3. 減少摩擦
4. 減少共振體
5. 密封音源(高頻)
6. 利用消音器
    - 散失型消音器(吸收)
    - 反應行銷音器(抗性)

### 控制工程：傳播途徑控制

1. 吸音材料的使用

    材料的吸音係數, 通常以六個頻率的吸音係數：125, 250, 500, 1000, 2000, 4000 Hz 表示。

    NRC 噪音減少係數為中間四個頻率(250, 500, 1000, 2000)的平均數。

2. 遙控式操作

3. 設置隔音屏

> STC（Sound Transmission Class）"噪音傳送等級"是一個用來表示裝修物件（例如牆壁、門、窗等）隔音能力的指標, 表示物件隔音的效果。STC 的數字越高, 隔音效果越好, 且 STC 是以分貝（dB）為單位。

## 震動作業 - 震動控制

彈簧的自然頻率$f_n$與靜變形之間的關係如下：

$ f\_{n}=\frac{5}{\sqrt{\delta_1[cm]}}[Hz]$

激動頻率與自然頻率比值與振動絕緣程度關係

| $\frac{f}{f_n}=R$ | 振動絕緣程度 |
| ----------------- | ------------ |
| <1.4              | 放大         |
| 1.4~3             | 可忽略       |
| 3~6               | 低           |
| 6~10              | 尚可         |
| >10               | 高           |

## 粉塵

適於下列各款之一之特定粉塵作業，雇主除於室內作業場所設置整體換氣裝置及於坑內作業場所設置第十一條第二項之換氣裝置外，並使各該作業勞工使用適當之呼吸防護具時，得不適用第六條之規定。

1. 於使用前直徑小於 30 公分之研磨輪從事作業時。
2. 使用搗碎或粉碎之最大能力每小時小於 20kg 之搗碎機或粉碎機從事作業時。
3. 使用篩選面積小於 700cm2 之篩選機從事作業時。
4. 使用內容積小於 18L 之混合機從事作業時。
